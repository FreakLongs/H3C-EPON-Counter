# EPON端口统计工具

一款专业的EPON设备端口信息采集与统计分析工具，支持SSH远程采集和本地文件分析两种模式。

## 功能特性

### 1. SSH远程采集模式
- 批量连接多台OLT设备
- 自动执行命令并采集端口数据
- 支持自定义SSH认证信息
- 实时显示采集进度和日志
- 支持中断停止采集

### 2. 本地文件分析模式
- 支持单个文件或批量文件处理
- 自动解析EPON端口数据
- 生成专业Excel统计报表
- 智能识别空闲PON口

### 3. Excel报表功能
- 美观的表格样式设计
- 多槽位数据统计
- 自动标记空闲端口
- 详细的备注说明

## 技术栈

- **GUI框架**: tkinter
- **SSH连接**: paramiko
- **Excel处理**: openpyxl
- **多线程**: threading
- **数据解析**: re (正则表达式)

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用说明

### SSH远程采集模式

1. 在"SSH远程采集"标签页中输入设备列表
   - 格式: `设备名-IP地址`
   - 示例: `山下湖-172.10.1.26`

2. 填写SSH认证信息
   - 用户名
   - 密码
   - 端口(默认22)

3. 选择输出目录

4. 点击"开始SSH采集"按钮

### 本地文件分析模式

1. 在"本地文件分析"标签页选择输入文件或文件夹

2. 选择输出目录

3. 点击"开始处理并生成Excel"按钮
